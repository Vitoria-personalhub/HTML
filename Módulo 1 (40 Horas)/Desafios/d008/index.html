Desafio 8: Navegação entre páginas

O que será desenvolvido:

O desafio proposto foi criar um site que apresente uma navegação entre páginas, seguindo as seguintes regras:

  1. Link 1-> página verde (pag002)
  2. Link 2-> página amarela (pag003)
  3. Na página verde, deverá existir um link para voltar para a página cinza
  4. Na página amarela, também deve conter um link para voltar para o índice

Esta é a página principal, o índice do seu site. É a página cinza. A partir dela você pode acessar as outras áreas principais através dos links a seguir:
