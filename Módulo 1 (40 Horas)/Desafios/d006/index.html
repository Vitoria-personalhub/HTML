Desafio 6: Exibição de Tags

O que será desenvolvido:

O desafio proposto foi juntar os conceitos de lista e títulos, com a exigência de, ao passar o mouse sobre um termo, o site mostre qual é a tag utilizada para fazer isso. Deve-se utilizar as seguintes categorias:

  1. Parágrafos
  2. Títulos
  3. Imagens
  4. Abreviações
  5. Códigos Fonte
  6. Listas Numeradas
  7. Listas Não Numeradas (com demarcadores)

Tags em HTML:

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deve ser utilizada em cada caso.

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Códigos Fonte
  • Listas Numeradas
  • Listas Não Numeradas (com demarcadores)
  • Citações
  • Endereços
  • Quebras de Linha
